# 自定义 API 配置说明

当前版本不再内置固定供应商选择。

你现在需要在设置中自行添加一个或多个 OpenAI 兼容接口，每个接口都可以单独配置：

- 接口名称
- Base URL
- API Key
- 模型列表

## 推荐用法

只需要先配置一个可用接口即可，不必一次性全部添加。

常见示例：

- 硅基流动：`https://api.siliconflow.cn/v1`
- 其他第三方 OpenAI 兼容服务：使用各自文档提供的 `v1` 接口地址

## 配置步骤

1. 打开对应服务商后台，创建 API Key。
2. 在应用设置中进入“自定义 API”。
3. 点击“添加接口”。
4. 填写接口名称、Base URL、API Key。
5. 在“模型列表”里一行填写一个模型 ID。
6. 保存设置。
7. 回到画布，在模型选择面板中选择刚配置的接口和模型。

## 模型列表示例

```text
Pro/google/nano-banana
black-forest-labs/FLUX.1-schnell
```

## 注意事项

- Base URL 不要多写结尾的 `/`，例如写 `https://api.siliconflow.cn/v1` 即可。
- 模型 ID 必须与服务商文档完全一致。
- 如果某个接口没有填写 API Key，该接口下的模型在画布中会提示先配置密钥。
- 同一个服务商可以添加多个接口，用于区分不同账号、不同线路或不同模型集合。
